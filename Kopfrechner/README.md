# Kopfrechentrainer

Sehr reduzierter Kopfrechen-Trainer fuer Smartphones.

## Inhalt

- Kleines und grosses 1x1 bis 20 x 20
- Divisionen aus dem kleinen und grossen 1x1
- Addition und Subtraktion mit 10er-Uebergang
- Lueckenaufgaben
- Keine Wiederholung innerhalb einer Runde
- Zeitbalken als Spielende-Mechanik

## Nutzung mit GitHub Pages

Repository auf GitHub hochladen und in den Repository-Einstellungen unter "Pages" die Veroeffentlichung aus dem Branch `main` aktivieren.
